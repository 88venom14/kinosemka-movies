# 🎬 CineSelect

Premium Cinema Booking Platform — система бронирования кинобилетов с минималистичным дизайном.

## 🚀 Быстрый старт

### Предварительные требования

- **Node.js** >= 18.x
- **PostgreSQL** >= 14.x
- **npm** или **yarn**

---

### 1. Настройка базы данных

Проект использует **PostgreSQL** в качестве основной базы данных.

#### Установка PostgreSQL

1. Скачайте установщик с [postgresql.org](https://www.postgresql.org/download/windows/)
2. Запустите установщик, выберите компоненты (PostgreSQL Server, pgAdmin, Command Line Tools)
3. При установке задайте пароль `admin` для пользователя `postgres`
4. После установки проверьте подключение:
   ```bash
   psql -U postgres -c "SELECT version();"
   ```

#### Создание базы данных

```bash
psql -U postgres -c "CREATE DATABASE moviesdatabase;"
```

> Параметры подключения по умолчанию:
> - Host: `127.0.0.1`
> - Port: `5432`
> - Database: `moviesdatabase`
> - User: `postgres`
> - Password: `admin`

Для изменения параметров отредактируйте `backend/src/db.js`.

#### Применение схемы

```bash
psql -U postgres -d moviesdatabase -f backend/database/schema.sql
```

#### Заполнение тестовыми данными

```bash
cd backend
npm run seed
```

Эта команда создаст:
- **8 фильмов** (Интерстеллар, Начало, Побег из Шоушенка, Тёмный рыцарь и др.)
- **3 зала** (Зал 1, Зал 2, VIP Зал)
- **Места** — 8 рядов × 12 мест (VIP ряды G, H с множителем цены ×1.5)
- **Сеансы** — на 7 дней вперёд, 5 сеансов в день (10:00–21:30)
- **3 тестовых пользователя** (см. раздел «Тестовые учётные данные»)

### 2. Установка зависимостей

```bash

cd backend
npm install

cd frontend

```

---

### 3. Настройка переменных окружения

Скопируйте `.env.example` в `.env` в директории `frontend/`:

```bash
cd frontend
cp .env.example .env
```

Основные переменные:

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `VITE_API_URL` | URL backend API | `http://localhost:5000/api` |
| `VITE_POISKKINO_TOKEN` | Токен PoiskKino API | — |
| `VITE_TMDB_API_KEY` | API ключ TMDB | — |

---

### 4. Запуск проекта

#### Backend (порт 5000)

```bash
cd backend
npm run dev
```

Сервер запустится на **http://localhost:5000** с автоматическим перезапуском при изменении файлов.

#### Frontend (порт 5173)

```bash
cd frontend
npm run dev
```

Приложение откроется на **http://localhost:5173**

## 📋 Описание

CineSelect — это премиальная платформа для онлайн-бронирования кинобилетов, разработанная с нуля для обеспечения бесшовного пользовательского опыта. Проект объединяет современный стек технологий с изысканным минималистичным дизайном, создавая атмосферу кинотеатра прямо в браузере.

### Ключевые особенности:

- **Стерильный минималистичный дизайн** — чёрный фон, точные границы, продуманная типографика и акцентные элементы в золотых тонах для VIP-зон
- **Плавные микро-взаимодействия** — анимации на базе Framer Motion обеспечивают отзывчивый и приятный интерфейс
- **Интуитивный процесс бронирования** — от выбора фильма до подтверждения бронирования за несколько кликов
- **Интерактивная схема зала** — визуальный выбор мест с разделением на обычные и VIP-зоны
- **Личный кабинет** — управление профилем, просмотр истории бронирований и отмена билетов
- **Mock API** — полная функциональность без необходимости запуска backend-сервера

## 🛠️ Технологический стек

### Frontend
- **React 18** + **TypeScript**
- **Vite** — сборка
- **Tailwind CSS** — стилизация
- **Framer Motion** — анимации
- **Zustand** — управление состоянием
- **React Router DOM** — роутинг
- **QRCode.react** — генерация QR-кодов

### Backend
- **Node.js** + **Express** — REST API
- **PostgreSQL** — база данных
- **Sequelize** — ORM
- **bcrypt** — хеширование паролей
- **jsonwebtoken** — аутентификация
- **AdminJS** — админ-панель
