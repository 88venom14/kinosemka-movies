# 🎬 CineSelect

Premium Cinema Booking Platform — система бронирования кинобилетов с минималистичным дизайном.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Запуск проекта

```bash
npm run dev
```

Проект откроется на **http://localhost:5173**

---

## 📋 Описание

CineSelect — это современная платформа для бронирования кинобилетов с акцентом на:
- Стерильный минималистичный дизайн (чёрный фон, точные границы, типографика)
- Плавные микро-взаимодействия и анимации
- Интуитивный процесс бронирования
- Личный кабинет с историей заказов

## 🛠️ Технологический стек

- **React 18** + **TypeScript**
- **Vite** — сборка
- **Tailwind CSS** — стилизация
- **Framer Motion** — анимации
- **Zustand** — управление состоянием
- **React Router DOM** — роутинг

## 📝 Функционал

### Для пользователей:
- ✅ Просмотр фильмов (сейчас в кино / скоро)
- ✅ Выбор сеанса
- ✅ Выбор мест (обычные / VIP)
- ✅ Бронирование билетов
- ✅ Личный кабинет
- ✅ История бронирований
- ✅ Отмена бронирования

### Особенности:
- Mock API для работы без backend
- Тестовые данные в `data/db.json`
- Локальное хранение состояния сессии

## 🎬 Тестовые данные

В проекте используются тестовые данные:

**Фильмы:**
- Дюна: Часть вторая
- Оппенгеймер
- Бэтмен
- Человек-паук: Через вселенные
- Стражи Галактики. Часть 3
- Джон Уик 4

**Сеансы:**
- Multiple showtimes для каждого фильма
- Разные цены (350-500 ₽)
- Разные залы (1-6)

**Места:**
- 8 рядов (A-H) × 12 мест = 96 мест на сеанс
- Ряды G, H — VIP (цена × 1.5)
- Некоторые места заняты
